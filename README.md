# fmd-adm

Rust bindings for illumos
[libfmd_adm](https://illumos.org/man/3LIB/libfmd_adm), the Fault
Management Daemon administrative library used to inspect and manage
illumos fault management state (modules, cases, resources, SERD engines,
and transports).

This repo contains two crates:

- **fmd-adm-sys** - raw FFI bindings generated by
  [bindgen](https://github.com/rust-lang/rust-bindgen)
- **fmd-adm** - idiomatic Rust wrapper with no raw pointers in the
  public API

These crates target illumos only and link against the system
`libfmd_adm`.

## Usage

```rust
use fmd_adm::FmdAdm;

let adm = FmdAdm::open()?;

for module in adm.modules()? {
    println!("{} v{} - {}", module.name, module.version, module.description);
}

for case in adm.cases(None)? {
    println!("{} - {}", case.uuid, case.code);
}
```

## Privileges

Most fmd operations require elevated privileges. Run with `pfexec` or
appropriate RBAC profiles.

## Testing

```
cargo nextest
```

Tests must be run on an illumos machine since they link against
`libfmd_adm`.
